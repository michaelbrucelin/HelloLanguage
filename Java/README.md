# Hello Java

## 安装jdk

```bash
apt install default-jdk  # default-jre
java --version   # 运行时
javac --version  # 编译器
```

## 创建java console项目

```bash
touch HelloWorld.java
vim HelloWorld.java
# public class HelloWorld {
#     public static void main(String[] args) {
#         System.out.println("Hello World!");
#     }
# }
```

## 运行项目

```bash
javac HelloWorld.java  # 编译
java HelloWorld        # 运行
Hello World!
```
