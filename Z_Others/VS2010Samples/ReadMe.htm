磁盘上的示例

Visual Basic、Visual C#、Visual C++ 的示例以及其他示例在安装 Visual Studio 的位置 以本地 .zip 文件的形式提供:

注意: 若要生成这些示例， 必须将 .zip 文件解压缩到您对其拥有写访问权的 文件夹中。
  1. 从本地 Samples 文件夹中选择 包含您要打开 的示例的 .zip 文件。
  2. 右击 .zip 文件，然后单击“全部提取...”将示例 解压缩到您所选的文件夹(如“My Documents”) 或解压缩到桌面。
