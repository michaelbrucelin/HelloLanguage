#### [878\. 第 N 个神奇数字](https://leetcode.cn/problems/nth-magical-number/)

难度：困难

一个正整数如果能被 `a` 或 `b` 整除，那么它是神奇的。

给定三个整数 `n` , `a` , `b` ，返回第 `n` 个神奇的数字。因为答案可能很大，所以返回答案 **对** `10<sup>9</sup> + 7` **取模** 后的值。

**示例 1：**

```
输入：n = 1, a = 2, b = 3
输出：2
```

**示例 2：**

```
输入：n = 4, a = 2, b = 3
输出：6
```

**提示：**

-   `1 <= n <= 10<sup>9</sup>`
-   `2 <= a, b <= 4 * 10<sup>4</sup>`
