#### [94\. 二叉树的中序遍历](https://leetcode.cn/problems/binary-tree-inorder-traversal/)

难度：简单

给定一个二叉树的根节点 `root` ，返回 _它的 **中序** 遍历_ 。

**示例 1：**

![](./Question0094.jpg)

```
输入：root = [1,null,2,3]
输出：[1,3,2]

```

**示例 2：**

```
输入：root = []
输出：[]

```

**示例 3：**

```
输入：root = [1]
输出：[1]

```

**提示：**

-   树中节点数目在范围 `[0, 100]` 内
-   `-100 <= Node.val <= 100`

**进阶:** 递归算法很简单，你可以通过迭代算法完成吗？
